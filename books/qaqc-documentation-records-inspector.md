# The QA/QC Documentation and Records Inspector

![QA/QC Documentation and Records Inspector Cover](/toolbox-to-table/assets/book-covers/qaqc-documentation-records-inspector-cover.jpg)

## Description

The QA/QC Documentation and Records Inspector provides practical guidance for professionals responsible for reviewing, organizing, verifying, and managing project quality records. Covering document control, turnover packages, material traceability records, inspection reports, test documentation, and final record compilation, the guide helps ensure project documentation is complete, accurate, and ready for client acceptance. Practical workflows and lessons learned support improved record management throughout the project lifecycle.

## Key Topics Covered

* Document control
* Material traceability
* Inspection reports
* Test records
* Turnover packages
* Quality documentation
* Record verification
* Final project closeout
