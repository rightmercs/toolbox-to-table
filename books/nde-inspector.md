# The NDE Inspector

![NDE Inspector Cover](/toolbox-to-table/assets/book-covers/nde-inspector-cover.jpg)

## Description

The NDE Inspector provides practical guidance for inspectors, technicians, and quality personnel involved in non-destructive examination activities. Covering common examination methods, procedural awareness, documentation review, and inspection oversight responsibilities, the guide helps professionals understand how NDE activities support overall quality verification. Practical field references and lessons learned help bridge the gap between technical procedures and real-world application.

## Key Topics Covered

* Visual testing
* Magnetic particle testing
* Liquid penetrant testing
* Ultrasonic testing
* Radiographic testing
* Procedure awareness
* Documentation review
* Inspection oversight
